# Software-Project-Management-University-Portal
This project is a Software Project Management (SPM) project created using Microsoft Project (MS Project).

The project focuses on planning, scheduling, and managing the development of a University Portal System. It includes project planning activities such as Work Breakdown Structure (WBS), task scheduling, project timeline, dependencies, milestones, resource allocation, and project tracking.

## Tools Used
- Microsoft Project (MS Project)

## Project Features
- Project Planning
- Work Breakdown Structure (WBS)
- Gantt Chart
- Task Scheduling
- Milestones
- Resource Management
- Project Timeline
- Dependency Management

This project was developed as part of the Software Project Management course to demonstrate project planning and management techniques using Microsoft Project.
